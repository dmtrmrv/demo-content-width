## Installation

This demo uses <a href="https://vitejs.dev/" target="_blank">Vite</a>. To run the demo locally follow these steps:

```bash
git clone git@github.com:dmtrmrv/demo-content-width.git demo-content-width
cd demo-content-width
npm install
npm run dev
```
